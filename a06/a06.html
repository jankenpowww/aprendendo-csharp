C# a06 - Formatando a Saída no Console

Uma vez que o processo de concatenar texto e variável torna-se complicado, temos outras alternativas mais viáveis e de fácil compreensão dentro do C#. Observe:



"O " + "jeito " + "convencional." + " (e outras formas mais interessantes de fazê-lo)"

Consideremos o programa abaixo que soma dois valores...


float n1, n2, total;

Console.Write("Digite um valor em R$ ");
n1 = float.Parse(Console.ReadLine());

Console.Write("Digite outro valor em R$ ");
n2 = float.Parse(Console.ReadLine());

total = n1 + n2;

Para concatenar esses valores com texto utilizando o sinal de mais, fazemos dessa forma:


...
Console.WriteLine("R$ " + n1 + " + " + "R$ " + n2 + " = " + "R$ " + total)

Não é tão complicado de entender em um primeiro momento, mas é inviável utilizar essa forma de formação de texto e variável a partir do momento em que temos que lidar com muitas variáveis, por exemplo.

Felizmente, temos alternativas mais simples e fáceis de fazer esse mesmo processo!

Formatação por índices: aqui, você informa entre chaves, vários índices diferentes de 0 até x número de variáveis que você quer exibir dentro de uma string. Depois, é só informar as variáveis correspondentes à ordem em que foi definida dentro da string e dos índices.


...

/*
Índice 0 = n1, índice 1 = n2, índice 3 = total.
Com a formatação de saída por índices, você basicamente informa a ordem
em que as variáveis vão aparecer dentro da string.
*/  

Console.WriteLine("R$ {0} + R$ {1} = R$ {2}", n1, n2, total)

Note como o código fica bem mais legível utilizando a formatação dessa forma.

Com a formatação de índices, você também pode aplicar tabulação na variável a ser exibida e ainda se tratando de valores numéricos, você pode converter o mesmo valor para um valor monetário local. Os valores formatados como valor monetário leva em consideração o localidade, nesse caso, o valor será formatado como real (BRL).


...

//Vai adicionar tabulação de 20 caracteres à esquerda da variável.
Console.WriteLine("R$ {0,20} <- n1", n1);
Console.WriteLine("R$ {0,20} <- n2", n2);

//O ':c' formata o valor à ser exibido em um valor monetário local.
Console.WriteLine("O resultado é {0:c}", total);

...

E ainda, temos uma forma ainda mais fácil, prática e legível de fazer a mesma coisa: formatando variável e string utilizando o cifrão. Nesta forma, você só precisa se preocupar com o conteúdo da string em si e informando o nome da variável diretamente entre as chaves!



...
Console.WriteLine($"{n1:c} + {n2:c} = {total:c}")
