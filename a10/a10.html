C# a10 - Enumeradores

O tal do enum

Em linguagens fortemente tipadas e orientadas a objeto como o próprio C#, os enumeradores, ou enums, são estruturas que organizam valores constantes finitos em um conjunto e que são enumerados. E apesar de ter seu conceito bastante próximo dos vetores, ou arrays, os enums se destacam justamente por essas duas qualidades mencionadas: serem valores constantes de um determinado conjunto que são enumerados. Os enums facilitam, sobretudo, a legibilidade do código, já que organizar elementos e declará-los no código é feito de forma literal.



Na prática...

Enums são especialmente definidos por um conjunto de valores finito. Assim como a abstração da teoria da programação orientada a objeto que se aproxima muito ao que observamos no mundo real, também podemos abstrair às coisas para entender esse tipo de estrutura.

E quais os exemplos que temos na vida real que remetem à essa descrição dos enums? Elementos que fazem parte ou compartilham de uma mesma característica e que são finitos? Temos os meses, as estações do ano e também os dias da semana. Todos eles fazem parte de um conjunto específico de alguma coisa, e além disso, compartilham de um elemento em comum que é bastante importante: eles podem ser enumerados.

E para atribuir à uma variável? Lembre-se que todos os elementos de um enum são do tipo definido pelo identificador. Ou seja, tanto Domingo quanto Quinta quanto todos os elementos deste enum são do tipo diasDaSemana!


...
    enum diasDaSemana {Domingo, Segunda, Terça, Quarta, Quinta, Sexta,
    Sábado}

    static void Main (string[] args)
    {
        diasDaSemana dia = diasDaSemana.Segunda;

        Console.WriteLine(dia);
    }
}

E o resultado no console é justamente mostrar 'Segunda'. Nada de conversão de strings a este ponto; se você quer mostrar o valor definido, é só mostrá-lo diretamente.

E quanto à parte mais importante que é eles serem enumerados, é possível converter o valor para seu valor numérico via typecasting.


...
    enum diasDaSemana {Domingo, Segunda, Terça, Quarta, Quinta, Sexta,
    Sábado}

    static void Main (string[] args)
    {
        diasDaSemana dia = diasDaSemana.Segunda;

        int numDia = (int) dia; //typecasting: convertendo para um valor inteiro.

        Console.WriteLine($"'{dia}' corresponde ao índice {numDia}
        dentro do enum.")
    }
}
Importante! É possível fazer o typecasting de um valor numérico utilizando o próprio tipo do enum.

O resultado no console será:


'Segunda' corresponde ao índice 1 dentro do enum.

...

Para recaptular: enumeradores nada mais é do que a estrutura que organiza em uma coleção várias constantes finitas que possuem valores correspondentes ao seu índice, valores esses obtidos por conversões explícitas feitas com typecast. No C#, ainda há a possibilidade de atribuir valores de número inteiro para um elemento dentro do índice, que retorna através do typecast o valor predefinido.
